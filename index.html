Free Code Camp Ziplines

This site is a live repository of my project solutions for the ziplines section of Free Code Camp

If you are not aware of what Free Code Camp is it is a Javascript centric web development course. It focuses on helping people learn to be full stack web designers using HTML5, CSS3, Bootstrap, Javascript, Jquery, Angular, and NodeJs. On top of the course work Free Code Camp also helps out non-profit organizations and helps new developers strengthen their portfolio and experince by matching the two together. Though relatively new, at the time of this writing 275 days old, the site already boasts more than 66 thousand members (affectionatly called campers by the site). The ciriculum is continuously improved as the site grows and it is a wonderful learning resource. The following projects I have made really have been thanks to this site.
